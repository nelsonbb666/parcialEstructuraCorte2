# Proyecto Pila y Cola

## 📌 Descripción
Aplicación en JavaScript que simula una pila (LIFO) y una cola (FIFO).

## 👨‍💻 Funcionalidades
- Pila: agregar y eliminar objetos
- Cola: agregar y eliminar números

## 🧠 Tecnologías
- JavaScript
